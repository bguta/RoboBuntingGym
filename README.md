# RoboBuntingGym
RL Gym Environment for Bilaterally Physical "Bunting" with a Robot

Requirements:
- Python 3.11


Steps to using this repository:
1. Create a virtual environment: `python -m venv .venv_gym`
2. Activate the virtual environment: `source .venv_gym/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
5. Run the training notebook: `bunting_sim_pce.ipynb`
6. Run the testing script: `python test_bunting_sim_pce.py`
